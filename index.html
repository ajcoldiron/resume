Portfolio

Home Page

Crypto Exchange
A responsive cryptocurrency dashboard that pulls real-time data from the CoinGecko API to display essential information on major cryptocurrencies like Bitcoin, Ethereum, and Tether. The dashboard includes a dynamic line chart that visualizes price fluctuations over customizable timeframes (week, month, year). To optimize performance and limit unnecessary API calls, I implemented Redux Persist, ensuring that data is stored locally and only updated when needed. Users can select and manage specific markets, viewing key metrics such as price, 24-hour high/low, and total supply.

Purchase Page

This page allows users to view and interact with specific cryptocurrency market pairs, such as ETH/BTC, displaying both their balance and market trends. The chart on the left visualizes the selected cryptocurrency pair's price movements over various timeframes (week, month, year), providing users with a clear overview of market behavior. On the right, users can select a currency to purchase, specify the purchase amount, and choose which cryptocurrency they want to use for the transaction. The user’s total exchange balances for multiple cryptocurrencies are displayed, allowing for easy tracking of their holdings. This interface offers a seamless and intuitive user experience, combining real-time data visualization with easy-to-use transaction functionality.

Assets Page

This page allows users to manage their cryptocurrency balances and transfer assets between their personal wallet and the exchange. It displays a real-time chart of the user’s total balance over time, giving a visual representation of their portfolio performance. Below the chart, users can select a cryptocurrency, choose between deposit or withdrawal options, and specify the transfer amount, enabling seamless transactions between wallets and exchanges. The interface also provides a detailed list of the user’s token balances, allowing for easy tracking and management of multiple cryptocurrencies.
